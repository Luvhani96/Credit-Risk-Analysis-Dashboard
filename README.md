💳 Credit Risk Analysis Dashboard (Excel)

Project Overview

This project analyzes customer credit data to identify patterns in loan risk behavior. Using Excel, I built an interactive dashboard that classifies customers into risk categories and explores key factors influencing credit default such as loan amount, age, duration, and loan purpose.
The goal of this project is to simulate real-world credit risk analysis used in banking and financial institutions to support data-driven lending decisions.

Objectives

Identify good vs bad credit risk customers
Analyze how loan amount affects risk levels
Explore relationships between age, purpose, and default risk
Build an interactive Excel dashboard for insights

 Tools & Skills I Used

Microsoft Excel
PivotTables & PivotCharts
Data Cleaning & Transformation
Conditional Logic (IF statements)
Data Segmentation
Dashboard Design
Business Intelligence Thinking

Key Features

> Credit Risk Classification (Good vs Bad borrowers)
> Loan Size Segmentation (Low, Medium, High)
> Age Group Analysis (Young, Adult, Senior)
> Loan Purpose Risk Breakdown
> Interactive dashboard with visual insights
> KPI metrics for quick decision-making

Dashboard Insights

Majority of customers fall under low-risk category
Higher loan amounts combined with longer durations increase default risk
Certain loan purposes (e.g., car and business loans) show higher risk levels
Younger customers tend to have slightly higher credit risk
Loan distribution shows clear segmentation between low and high-risk groups
